# Jekyll-blog

Learning to build a project with Jekyll

## Getting started

- Install all the dependencies with this command

```markdown
bundle install
```

## Create a new project

For example for a blog, you can use

```markdown
bundle exec jekyll new blog
```

## Launch the server

```markdown
bundle exec jekyll serve
```
